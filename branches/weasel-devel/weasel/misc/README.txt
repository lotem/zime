# 說明書

【小狼毫】 
預覽版 20100822
核心版本 3.4

http://zime.googlecode.com/

中州式恕堂 版權所無

------------------------------------------------------------

安裝輸入法

weasel-preview-20100822.zip

本品採用綠色安裝方式，請由以上軟件包解包出 weasel 文件夾，
執行其中的安裝腳本：

Windows XP 系統用
> xp_install.bat

Windows 7 系統用
> win7_install.bat

------------------------------------------------------------

安裝輸入方案

在 weasel\data 文件夾內執行腳本：
> populate_db.bat

產出數據庫文件位於 %USERPROFILE%\.ibus\zime\zime.db 。內含：

  * 拼音（簡體中文）
  * 雙拼
  * 宮保拼音，自家釀造的標準鍵盤並擊打字法
  * 標調拼音，佛振推介！
  * 粵拼／Jyutping

更多輸入方案及定製方法見於
http://code.google.com/p/zime/wiki/ZIMEData

------------------------------------------------------------

使用輸入法

選取輸入法通知區的【中】字樣圖標，開始打字

要切換輸入方案，隨時按 Ctrl+` 調出「方案選單」

各輸入方案對按鍵的功用有自己的定義。常見的設置為：
  * 字母做編碼，「'」做隔音符號，數字鍵選詞
  * 空格鍵確認當前高亮顯示的候選詞，Enter鍵使編碼串上屏
  * 「`」做西文模式前導鍵
  * 「,」、「.」前後換頁
  * 上、下鍵在當前一組候選詞中切換，左、右鍵改變選詞範圍
